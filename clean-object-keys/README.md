# clean-object-keys

🧹 A tiny JavaScript utility to remove `null`, `undefined`, and `""` (empty string) values from an object.

## 📦 Installation

```bash
npm install clean-object-keys
```

const { cleanObject } = require("clean-object-keys");

  name: "Manu",
  email: "",
  phone: undefined,
  age: null,
  country: "India",
